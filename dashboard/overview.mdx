# Dashboard Overview

The Edge dashboard at [app.edge.bh](https://app.edge.bh) gives you full visibility into your API usage, credit balance, and team management.

## Home

When you sign in, the dashboard shows:

- **Credit Balance** — Your current available credits
- **Calls Today** — API requests made today
- **This Month** — Total requests this month
- **Credits Used** — Credits consumed this month

Below the stats, the **Getting Started** checklist guides you through:

1. Create your organization
2. Create your first API key
3. Make your first API request

The **API Usage** chart shows your request volume over the last 30 days, with a per-service breakdown.
