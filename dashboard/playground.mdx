# Playground

The Playground lets you test any Edge API service directly from your dashboard — no code required.

## How it works

1. Go to **Playground** in your dashboard
2. Select a service from the dropdown (IBAN Validation, Sanctions Screening, etc.)
3. Select an API key to use
4. Enter the required parameters
5. Click **Send Request**

The Playground shows:
- The full request URL
- Response body with syntax highlighting
- Response status code
- Response time in milliseconds
- Credits consumed

## Sample data

Each service includes sample data buttons (country flags for IBAN, pre-filled names for sanctions) so you can test immediately without entering your own data.

<Note>
Playground requests consume credits just like regular API calls. They appear in your Request Logs.
</Note>
