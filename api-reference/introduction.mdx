# API Reference

Base URL for all API requests:

```
https://api.edge.bh/api/v1
```

## Authentication

All service endpoints require an API key passed as a Bearer token in the `Authorization` header:

```bash
curl -X GET "https://api.edge.bh/api/v1/services/iban/validate/GB29NWBK60161331926819" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

## Response format

All responses are JSON. Successful responses return `200` with the data. Error responses include an `error` field.
