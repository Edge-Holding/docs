# Countries by Calling Code

<Info>**Credits:** 1 per call</Info>

Returns all countries that use the specified phone calling code.

## Request

```bash
GET /services/country/calling-code/{code}
```

<ParamField path="code" type="string" required>
  Phone calling code (without `+`). Example: `973`, `1`, `44`.
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/country/calling-code/973" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.edge.bh/api/v1/services/country/calling-code/973",
    headers={"Authorization": "Bearer sk_live_your_key_here"}
)
```

</CodeGroup>

### Response fields

<ResponseField name="calling_code" type="string">
  The calling code that was searched (with `+` prefix).
</ResponseField>

<ResponseField name="countries" type="array">
  List of countries using this calling code. Each country uses the full format (same fields as [Country Lookup](/api-reference/country/lookup)).
</ResponseField>

<ResponseField name="total" type="integer">
  Total number of matching countries.
</ResponseField>
