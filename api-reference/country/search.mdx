# Search Countries

<Info>**Credits:** 1 per call</Info>

Search for countries by common name, official name, native name, or country code (alpha-2 or alpha-3). Partial match supported.

## Request

```bash
GET /services/country/search
```

<ParamField query="q" type="string" required>
  Search query (minimum 2 characters).
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/country/search?q=bahrain" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.edge.bh/api/v1/services/country/search",
    params={"q": "bahrain"},
    headers={"Authorization": "Bearer sk_live_your_key_here"}
)
```

</CodeGroup>
