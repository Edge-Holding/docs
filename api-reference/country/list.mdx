# List Countries

<Info>**Credits:** 1 per call</Info>

Returns a list of all 250 countries with basic information, sorted alphabetically by name.

## Request

```bash
GET /services/country/list
```

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/country/list" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.edge.bh/api/v1/services/country/list",
    headers={"Authorization": "Bearer sk_live_your_key_here"}
)
```

</CodeGroup>

### Response fields

<ResponseField name="countries" type="array">
  List of countries, each containing:
  <Expandable>
    <ResponseField name="code" type="string">ISO 3166-1 alpha-2 code.</ResponseField>
    <ResponseField name="code3" type="string">ISO 3166-1 alpha-3 code.</ResponseField>
    <ResponseField name="name" type="string">Common country name.</ResponseField>
    <ResponseField name="official_name" type="string">Official country name.</ResponseField>
    <ResponseField name="flag" type="string">Flag emoji.</ResponseField>
    <ResponseField name="region" type="string">Geographic region.</ResponseField>
    <ResponseField name="population" type="integer">Country population.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="integer">
  Total number of countries returned.
</ResponseField>
