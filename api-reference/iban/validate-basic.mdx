# Validate IBAN (Basic)

<Info>**Credits:** 1 per call</Info>

A lightweight validation that checks the IBAN format and checksum without looking up bank details. Returns country info, SEPA scheme support, and the extracted BBAN and account number.

## Request

<ParamField path="iban" type="string" required>
  The IBAN to validate.
</ParamField>

```bash
GET /services/iban/validate-basic/{iban}
```

Both `GET` and `POST` methods are supported.

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/iban/validate-basic/DE89370400440532013000" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.edge.bh/api/v1/services/iban/validate-basic/DE89370400440532013000",
    headers={"Authorization": "Bearer sk_live_your_key_here"}
)
data = response.json()
```

</CodeGroup>

## Response

<ResponseField name="result" type="integer">
  `200` if the IBAN is valid, `400` if it is not.
</ResponseField>

<ResponseField name="message" type="string">
  `"IBAN is valid"` or `"IBAN is not valid"`.
</ResponseField>

<ResponseField name="validations" type="array">
  Array of validation checks performed. Each item has `code` and `message`.
</ResponseField>

<ResponseField name="data" type="object">
  Parsed IBAN data.
  <Expandable>
    <ResponseField name="iban" type="string">The normalized IBAN (spaces removed, uppercased).</ResponseField>
    <ResponseField name="country_code" type="string">ISO 3166-1 alpha-2 country code.</ResponseField>
    <ResponseField name="country_name" type="string | null">Country name.</ResponseField>
    <ResponseField name="currency_code" type="string | null">ISO 4217 currency code.</ResponseField>
    <ResponseField name="sepa" type="object">
      SEPA scheme support details.
      <Expandable>
        <ResponseField name="member" type="boolean">Whether the country is a SEPA member.</ResponseField>
        <ResponseField name="sepa_credit_transfer" type="boolean">Supports SEPA Credit Transfer.</ResponseField>
        <ResponseField name="sepa_direct_debit" type="boolean">Supports SEPA Direct Debit.</ResponseField>
        <ResponseField name="sepa_cor1" type="boolean">Supports SEPA COR1.</ResponseField>
        <ResponseField name="sepa_b2b" type="boolean">Supports SEPA B2B.</ResponseField>
        <ResponseField name="sepa_card_clearing" type="boolean">Supports SEPA Card Clearing.</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="bban" type="string">Basic Bank Account Number.</ResponseField>
    <ResponseField name="bank_account" type="string | null">Account number extracted from the IBAN.</ResponseField>
  </Expandable>
</ResponseField>
