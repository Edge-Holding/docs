# Bulk Validate IBANs

<Info>**Credits:** 2 per IBAN in the request</Info>

Validate between 5 and 100 IBANs in a single API call. Each IBAN gets full validation with bank details, SEPA data, and validation checks.

## Request

```bash
POST /services/iban/bulk-validate
```

<ParamField body="ibans" type="string[]" required>
  Array of IBAN strings. Minimum 5, maximum 100.
</ParamField>

<CodeGroup>

```bash cURL
curl -X POST "https://api.edge.bh/api/v1/services/iban/bulk-validate" \
  -H "Authorization: Bearer sk_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "ibans": [
      "GB29NWBK60161331926819",
      "DE89370400440532013000",
      "FR7630006000011234567890189",
      "ES9121000418450200051332",
      "IT60X0542811101000000123456"
    ]
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.edge.bh/api/v1/services/iban/bulk-validate",
    headers={
        "Authorization": "Bearer sk_live_your_key_here",
        "Content-Type": "application/json"
    },
    json={
        "ibans": [
            "GB29NWBK60161331926819",
            "DE89370400440532013000",
            "FR7630006000011234567890189",
            "ES9121000418450200051332",
            "IT60X0542811101000000123456"
        ]
    }
)
```

</CodeGroup>

## Response

<ResponseField name="results" type="array">
  Array of full validation results. Each item has the same structure as the [Validate IBAN (Full)](/api-reference/iban/validate) response: `valid`, `iban`, `bank_data`, `sepa_data`, `validations`, and `errors`.
</ResponseField>

<ResponseField name="count" type="integer">
  Number of IBANs processed.
</ResponseField>
