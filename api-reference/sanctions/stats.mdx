# Sanctions Database Stats

<Info>**Credits:** Free</Info>

Returns statistics about the current sanctions database, including total entities, breakdown by entity type, and index term count.

## Request

```bash
GET /services/sanctions/stats
```

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/sanctions/stats" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

</CodeGroup>

## Response

```json
{
  "total_entries": 1200000,
  "types": {
    "Entity": 650000,
    "Individual": 450000,
    "Vessel": 80000,
    "Aircraft": 20000
  },
  "index_terms": 3500000
}
```
