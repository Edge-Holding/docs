# Validate Phone

<Info>**Credits:** 1 per call</Info>

Validates a phone number and returns formatting information, country detection, line type (mobile/landline), and E.164 formatted number.

## Request

```bash
GET /services/phone/validate/{phone}
```

Both `GET` and `POST` methods are supported. For `POST`, pass the phone number in the request body.

<ParamField path="phone" type="string" required>
  The phone number to validate. Include country calling code (with `+`) for best results.
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/phone/validate/+97317000000" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.edge.bh/api/v1/services/phone/validate/+97317000000",
    headers={"Authorization": "Bearer sk_live_your_key_here"}
)
```

</CodeGroup>

## Response

```json
{
  "phone": "+97317000000",
  "valid": true,
  "e164": "+97317000000",
  "international": "+973 17000000",
  "national_number": "17000000",
  "calling_code": "+973",
  "country_code": "BH",
  "type": "landline",
  "checks": {
    "format": true,
    "country_detected": true,
    "valid_length": true
  },
  "reason": null
}
```

### Response fields

<ResponseField name="phone" type="string">
  The original phone number as provided.
</ResponseField>

<ResponseField name="valid" type="boolean">
  Whether the phone number is valid (country detected and length matches expected format).
</ResponseField>

<ResponseField name="e164" type="string | null">
  E.164 formatted number (e.g., `+97317000000`). Recommended format for storage. `null` if country could not be detected.
</ResponseField>

<ResponseField name="international" type="string | null">
  International format with space after calling code (e.g., `+973 17000000`). `null` if country could not be detected.
</ResponseField>

<ResponseField name="national_number" type="string | null">
  The national number without the calling code. `null` if country could not be detected.
</ResponseField>

<ResponseField name="calling_code" type="string | null">
  The detected calling code with `+` prefix (e.g., `+973`). `null` if not detected.
</ResponseField>

<ResponseField name="country_code" type="string | null">
  ISO 3166-1 alpha-2 country code (e.g., `BH`). `null` if country could not be detected.
</ResponseField>

<ResponseField name="type" type="string">
  Line type: `mobile`, `landline`, or `unknown`.
</ResponseField>

<ResponseField name="checks" type="object">
  Individual validation check results.
  <Expandable>
    <ResponseField name="format" type="boolean">Whether the number has between 7 and 15 digits.</ResponseField>
    <ResponseField name="country_detected" type="boolean">Whether a country was detected from the calling code.</ResponseField>
    <ResponseField name="valid_length" type="boolean">Whether the national number length matches the expected length for the detected country.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="reason" type="string | null">
  Human-readable explanation when the number is invalid. `null` when valid.
</ResponseField>
