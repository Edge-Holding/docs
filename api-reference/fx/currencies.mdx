# List Currencies

<Info>**Credits:** 1 per call</Info>

Returns a map of all supported currency codes.

## Request

```bash
GET /services/fx/currencies
```

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/fx/currencies" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

</CodeGroup>

## Response

```json
{
  "currencies": {
    "AED": "AED",
    "BHD": "BHD",
    "EUR": "EUR",
    "GBP": "GBP",
    "SAR": "SAR",
    "USD": "USD"
  }
}
```

<Note>166+ currencies are supported. The response includes all available currency codes sorted alphabetically.</Note>
