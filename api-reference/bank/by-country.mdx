# Banks by Country

<Info>**Credits:** 1 per call</Info>

Returns a list of banks in the specified country with pagination.

## Request

```bash
GET /services/bank/country/{country}
```

<ParamField path="country" type="string" required>
  ISO 3166-1 alpha-2 country code. Example: `GB`, `DE`, `BH`.
</ParamField>

<ParamField query="search" type="string">
  Filter banks by name.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number.
</ParamField>

<ParamField query="per_page" type="integer" default="25">
  Results per page.
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/bank/country/GB?search=national&per_page=10" \
  -H "X-Api-Key: sk_live_your_key_here"
```

</CodeGroup>

## Response

```json
{
  "country": "GB",
  "banks": [
    {
      "bank_name": "NATIONAL WESTMINSTER BANK PLC",
      "bic": "NWBKGB2L",
      "city": "London"
    }
  ],
  "total": 1,
  "page": 1,
  "per_page": 10
}
```
