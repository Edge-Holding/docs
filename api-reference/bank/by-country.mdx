# Banks by Country

<Info>**Credits:** 1 per call</Info>

Returns a paginated list of banks in the specified country. Supports filtering by bank name, BIC, or city.

## Request

```bash
GET /services/bank/country/{country}
```

<ParamField path="country" type="string" required>
  ISO 3166-1 alpha-2 country code. Example: `GB`, `DE`, `BH`.
</ParamField>

<ParamField query="search" type="string">
  Filter banks by name, BIC, or city.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number.
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Results per page (max 100).
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/bank/country/GB?search=national&per_page=10" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

</CodeGroup>
