# Supported Countries

<Info>**Credits:** 1 per call</Info>

Returns a list of all countries that have bank directory data, with a count of banks per country.

## Request

```bash
GET /services/bank/countries
```

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/bank/countries" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

</CodeGroup>

## Response

```json
{
  "countries": [
    { "country_code": "BH", "bank_count": 28 },
    { "country_code": "DE", "bank_count": 380 },
    { "country_code": "FR", "bank_count": 320 },
    { "country_code": "GB", "bank_count": 450 }
  ],
  "total_countries": 63,
  "total_banks": 3000
}
```
