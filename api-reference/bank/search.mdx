# Search Banks

<Info>**Credits:** 1 per call</Info>

Search for banks across all countries by name or BIC code. Query must be at least 2 characters.

## Request

```bash
GET /services/bank/search
```

<ParamField query="q" type="string" required>
  Search query (bank name or BIC). Must be at least 2 characters.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number.
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Results per page (max 100).
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/bank/search?q=deutsche+bank" \
  -H "Authorization: Bearer sk_live_your_key_here"
```

</CodeGroup>

## Response

```json
{
  "search": "deutsche bank",
  "results": [
    {
      "bic": "DEUTDEFF",
      "bank_code": "50070010",
      "bank": "Deutsche Bank AG",
      "country_code": "DE",
      "city": "Frankfurt am Main"
    }
  ],
  "total": 1,
  "page": 1,
  "per_page": 20
}
```
