# Commercial Registry Lookup

<Info>**Credits:** 10 per call</Info>

Retrieve full details of a commercial registration in Bahrain, including company information, shareholders, board members, branches, and activities.

## Request

```bash
GET /services/cr/bh/lookup/{crNumber}
```

<ParamField path="crNumber" type="string" required>
  The commercial registration number.
</ParamField>

<ParamField query="branch" type="string">
  Specific branch number to look up.
</ParamField>

<ParamField query="lang" type="string" default="EN">
  Response language: `EN` or `AR`.
</ParamField>

<Note>Currently only Bahrain (`bh`) is supported. More countries coming soon.</Note>

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/cr/bh/lookup/12345" \
  -H "X-Api-Key: sk_live_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.edge.bh/api/v1/services/cr/bh/lookup/12345",
    headers={"X-Api-Key": "sk_live_your_key_here"}
)
```

</CodeGroup>
