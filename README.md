# Edge Documentation

Documentation for the [Edge API](https://edge.bh) — one API key for IBAN validation, sanctions screening, exchange rates, and 9 more financial data services.

Built with [Mintlify](https://mintlify.com).

## Development

```bash
# Install CLI
npm i -g mint

# Run locally
npx mint dev
```

Preview at `http://localhost:3000`.

## Deployment

Pushes to `main` auto-deploy via the Mintlify GitHub app.
