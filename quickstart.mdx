# Quickstart

Get started with Edge in three steps: create an account, get your API key, and make your first API call.

## 1. Create an account

Sign up at [app.edge.bh](https://app.edge.bh). You'll get 500 free credits — no credit card required.

Once signed up, you'll be prompted to create an organization. All API keys, credits, and usage are scoped to your organization.

## 2. Get your API key

Go to **API Keys** in your dashboard and click **Create API Key**. Give it a name and optionally set an expiration date.

<Warning>
Your API key is shown once on creation. Copy it immediately and store it securely — you won't be able to see it again.
</Warning>

Your key will look like: `sk_live_abc123...`

## 3. Make your first API call

Validate an IBAN with a single request:

<CodeGroup>

```bash cURL
curl -X GET "https://api.edge.bh/api/v1/services/iban/validate/GB29NWBK60161331926819" \
  -H "X-Api-Key: sk_live_your_key_here"
```

```javascript JavaScript
const response = await fetch(
  "https://api.edge.bh/api/v1/services/iban/validate/GB29NWBK60161331926819",
  {
    headers: {
      "X-Api-Key": "sk_live_your_key_here",
    },
  }
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.edge.bh/api/v1/services/iban/validate/GB29NWBK60161331926819",
    headers={"X-Api-Key": "sk_live_your_key_here"}
)
print(response.json())
```

```php PHP
$response = Http::withHeaders([
    'X-Api-Key' => 'sk_live_your_key_here',
])->get('https://api.edge.bh/api/v1/services/iban/validate/GB29NWBK60161331926819');

$data = $response->json();
```
